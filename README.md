Global Aviation Performance Analysis ✈️
===========
Project Overview
---
This project provides a comprehensive analysis of global aviation trends, focusing on on-time performance (OTP), flight volumes, and the operational efficiency of major hubs and carriers. By leveraging historical flight data, we aim to identify patterns in delays and provide actionable insights for stakeholders in the aviation industry.

Key Objectives
---
Trend Analysis: Evaluate year-over-year growth in global flight traffic.

Delay Attribution: Categorize and quantify the primary causes of flight delays (weather, NAS, late-arriving aircraft, etc.).

Efficiency Benchmarking: Compare the performance of the top 10 global airlines and airports.

Predictive Insights: Modeling potential delay risks based on seasonal data.

Data Architecture & Workflow
---
The project follows a standard data science pipeline, from ingestion to visualization.

Data Collection: Sourced from kaggle global aviation dataset. 

Preprocessing: Cleaning null values, handling time-zone conversions, and normalizing carrier codes.

Visualization: Built interactive dashboards to display regional performance.

Tech Stack
---
Language    :    Python (Pandas, NumPy)

Visualization :  Matplotlib, Seaborn,Tableau/PowerBI

Analysis      :  Scikit-Learn 

Environment   :  Jupyter Notebooks / VS Code
